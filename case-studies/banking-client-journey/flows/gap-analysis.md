# Gap Analysis (AS-IS vs TO-BE)

| Area         | AS-IS      | TO-BE                     |
| ------------ | ---------- | ------------------------- |
| Workflow     | Sequential | Partially parallel        |
| Ownership    | Fragmented | Clearly defined per stage |
| Visibility   | Low        | Centralised tracking      |
| Coordination | Manual     | Structured                |
| Efficiency   | Low        | Improved                  |

---

## Key Transformation

The improvement comes from restructuring coordination and ownership, not replacing systems.
